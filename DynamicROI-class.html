Class "DynamicROI"

Class "DynamicROI"

Description

Contains information about ROIs that change position or shape over time

Slots

.Data
a list of polygon coordinates
times
a data frame where each row contains the begin and end time for each ROI position from the main list
ID
an object of class "character"; a text based ID
layer
an object of class "numeric"; the layer (z-order / depth) of the ROI
parent
an object of class "character"; the id of the parent ROI
