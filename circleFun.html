Circle Function

Usage

circleFun(center = c(0, 0), diameter = 1, npoints = 50, round = FALSE)

Arguments

center
the center point of the circle
diameter
the diameter of the circle
npoints
the number of points to use in the circle
round
if TRUE, points will be rounded to nearest integer

Circle Function

Value

a data frame with columns for the x and y component of each point in the circle

Description

Create a circle from comprised of limited set of points.

Examples

require(ggplot2)


Loading required package: ggplot2

c <- circleFun(center = c(5, 5), diameter = 2)
ggplot(c) + geom_point(aes(x=x,y=y)) + coord_equal()
