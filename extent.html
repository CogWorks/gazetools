Extent (range) of fixations

Usage

extent(x)

Arguments

x
coordinate vectors of points. This can be specified as a 2-column matrix x, a list x with two components

Value

a list with two values, the horizontal (x) and vertical (y) range of points

Description

Calculates the distance between fixations in the horizontal and vertical meridians

Examples

data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl, smi_time, 500, 1680, 1050, 473.76, 296.1, smi_ezl, smi_exl, smi_eyl))
d.c <- classify.V(d.pva@v)
d.f <- getFixations(d.c, d.pva)
d.extent <- extent(d.f[,c("x","y")])
str(d.extent)
