classify-class {gazetools}R Documentation

Class "classify"

Description

A class to hold gaze data classifications

Slots

.Data:

vector of class "character", containing the gaze sample classification

fixation_ids:

vector of class "numeric", containing unique fixation ids

saccade_ids:

vector of class "numeric", containing unique saccade ids

algorithm:

the algorithm used to classify the gaze data

thresholds:

the threshold settings for the classification algorithm
