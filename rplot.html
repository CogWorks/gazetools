Raster Plot

Usage

rplot(raster, xlim = NULL, ylim = NULL, rois = NULL, ...)

Arguments

raster
a rasterized image
xlim
a vector of length 2 that contains the min and max horizontal bounds of the image
ylim
a vector of length 2 that contains the min and max vertical bounds of the image
rois
an object of class "ROIs", used to overlay ROI polygons
...
extra params passed to geom_path, only used when rois!=NULL

Raster Plot

Description

A wrapper around a few ggplot2 functions that creates a base plot layer out of a raster image. Coordinates are restricted to values within the raster image and the aspect ratio is fixed.

Examples

data(logo)
rplot(as.raster(logo))
