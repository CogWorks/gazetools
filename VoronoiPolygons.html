VoronoiPolygons

Usage

VoronoiPolygons(x, rw)

Arguments

x
a matrix of x,y cooridinates
rw
the coordinates of the corners of the rectangular window enclosing the voronoi cells, in the order (xmin, xmax, ymin, ymax).

VoronoiPolygons

Value

an object of class VoronoiPolygons

Description

Creates an object of class VoronoiPolygons

Examples

data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl,
                       500, 1680, 1050, 473.76, 296.1,
                       smi_ezl, smi_exl, smi_eyl))
d.c <- classify.V(d.pva@v)
d.f <- getFixations(d.c, d.pva)
d.vp <- VoronoiPolygons(d.f[,c("fixation.x","fixation.y")], c(315,1365,0,1050))
str(d.vp)
