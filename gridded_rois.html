Gridded ROIs

Usage

gridded_rois(xmin, xmax, ymin, ymax, ncol, nrow)

Arguments

xmin
the lower limit of the x axis
xmax
the upper limit of the x axis
ymin
the lower limit of the y axis
ymax
the upper limit of the y axis
ncol
the number of columns in the grid
nrow
the number of rows in the grid

Value

an object of class gridded_rois

Description

Creates a grid of ROIs over a given 2-D region

Examples

rois <- gridded_rois(315,1365,0,1050,nrow=11,ncol=11)
