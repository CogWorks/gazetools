Get Glissades

Usage

getGlissades(class, dpva)

Arguments

class
an object of class classify
dpva
an object of class pva

Description

Extracts the start point, end point, distance and duration of glissades from classify and pva objects

Examples

data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl,
                       500, 1680, 1050, 473.76, 296.1,
                       smi_ezl, smi_exl, smi_eyl, pupil=smi_dyl))
d.c <- classify.VI(d.pva@v, sigma=4, blinks=d.pva@blinks)
d.f <- getGlissades(d.c, d.pva)
d.f
