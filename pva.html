Compute Velocity and Acceleration Profiles for Raw Gaze Data

Usage

pva(x, y, samplerate, rx, ry, sw, sh, ez, ex = 0, ey = 0, order = 2, window = 11, pupil = NULL, blinkFUN = "detect_blinks.PV", ...)

Arguments

x
the x coordinate of a point on a screen (pixels)
y
the y coordinate of a point on a screen (pixels)
samplerate
the samplerate of the eyetracker
rx
the x resolution of the monitor (pixels)
ry
the y resolution of the monitor (pixels)
sw
the physical screen width (mm)
sh
the physical screen height (mm)
ez
the perpendicular distance from the viewer to the screen (mm)
ex
the horizontal offset of the viewer from screen center (mm)
ey
the vertical offset of the viewer from screen center (mm)
order
the Savitzky-Golay filter order
window
the Savitzky-Golay filter window length (must be odd)
pupil
a vector of class numeric; the vertical diameter of the pupil or a pupil status signal
blinkFUN
the name of the blink detection algorithm to use
...
arguments to be passed to blinkFUN

Value

an object of class pva

Description

Takes standard eye tracker gaze information and computes smoothed position, velocity and acceleration profiles using a Savitzky-Golay filter.

Examples

data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl, 
                       500, 1680, 1050, 473.76, 296.1, 
                       smi_ezl, smi_exl, smi_eyl))
str(d.pva)
