pva(x, y, samplerate, rx, ry, sw, sh, ez, ex = 0, ey = 0, order = 2, window = 11, vt = 1000, at = 1e+05, pupil = NULL, blinkFUN = "detect_blinks.SW", approxFUN = "na.spline", ...)

ry
the y resolution of the monitor (pixels)

the physical screen width (mm)
sh
the physical screen height (mm)

the perpendicular distance from the viewer to the screen (mm)
ex

the Savitzky-Golay filter order

the Savitzky-Golay filter window length (must be odd)
vt

max acceleration threshold

blinkFUN

approxFUN

...

Compute Velocity and Acceleration Profiles for Raw Gaze Data

Value

an object of class pva

Description

Takes standard eye tracker gaze information and computes smoothed position, velocity and acceleration profiles using a Savitzky-Golay filter.

Examples

data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl,
                       500, 1680, 1050, 473.76, 296.1,
                       smi_ezl, smi_exl, smi_eyl))
