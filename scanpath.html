Scanpath

Usage

scanpath(x, y = NULL, rois, nearest = FALSE)

Arguments

x,y
the x and y coordinates of a set of points. Alternatively, a single argument x can be provided
rois
an object of class Polygons
nearest
if TRUE fixations not on an ROI will be moved to the nearest ROI

Value

an object of class Scanpath

Description

Determines the scanpath given a set of fixations and a list of polygon ROIs
