Gaze Data Classification (velocity-iterative)

Usage

classify.VI(v, vt = 100, sigma = 6, samplerate = 500, min.fix = 0.04, min.sac = 0.01, blinks = FALSE, a = 0.7, b = (1 - a))

Arguments

v
a vector of instantaneous velocity values
vt
a velocity threshold
sigma
number of standard deviations used for upper saftey margin for peak velocity threshold, saccade velocity threshold is half this value, this parameter needs to be tuned for each eyetracker/dataset combination
samplerate
the samplerate in Hz of the eyetracker
min.fix
minimum fixations duration (in seconds)
min.sac
minimum saccade duration (in seconds)
blinks
vector of class "logical", TRUE if samples belong to a blink
a
weight of saccade onset threshold for determining saccade offset threshold
b
weight of noise threshold for determining saccade offset threshold

Gaze Data Classification (velocity-iterative)

Value

an object of class classify

Description

Classifies gaze data into saccades and fixations using an iteratively determined velocity threshold.
