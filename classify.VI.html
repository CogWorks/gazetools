Gaze Data Classification (velocity-iterative)

Usage

classify.VI(v, vt = 100, sigma = 6, blinks = NULL)

Arguments

v
a vector of instantaneous velocity values
vt
a velocity threshold
sigma
the number of standard deviations between fixation and saccade velocities
blinks
vector of class "logical", TRUE if samples belong to a blink

Value

an object of class classify

Description

Classifies gaze data into saccades and fixations using an iteratively determined velocity threshold.

Examples

# Classification ignorning blinks
data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl,
                       500, 1680, 1050, 473.76, 296.1,
                       smi_ezl, smi_exl, smi_eyl))
d.c <- classify.VI(d.pva@v)
str(d.c)
