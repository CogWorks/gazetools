Get Scanpath

Usage

getScanpath(x, y = NULL, begin = NULL, end = NULL, rois, nearest = FALSE, firstOnly = TRUE)

Arguments

x,y
the x and y coordinates of a set of points. Alternatively, a single argument x can be provided
begin,end
the begin and end timestamp of each fixation, only used with DynamicROIs
rois
an object of class ROIs or DynamicROIs
nearest
if TRUE fixations not on an ROI will be moved to the nearest ROI
firstOnly
if TRUE only DynamicROIs present at the start of a fixation are reported

Get Scanpath

Value

an object of class Scanpath

Description

Determines the scanpath given a set of fixations and a list of polygon ROIs
