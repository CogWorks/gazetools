Class "ROI"

Class "ROI"

Description

An extension of the class "Polygon" that additionally contains a text ID

Slots

ID
an object of class "character"; a text based ID
center
an object of class "numeric"; the center coordinates of the ROI
labpt
an object of class "numeric"; an x, y coordinate pair forming the label point of the polygon
area
an object of class "numeric"; the planar area of the polygon, does not respect projection as objects of this class have no projection defined
hole
an object of class "logical"; does the polygon seem to be a hole
ringDir
an object of class "integer"; the ring direction of the ring (polygon) coordinates, holes are expected to be anti-clockwise
coords
an object of class "matrix"; coordinates of the polygon; first point should equal the last point
