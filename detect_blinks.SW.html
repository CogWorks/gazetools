Detect Blinks (static window)

Usage

detect_blinks.SW(pupil, samplerate, pad = 50)

Arguments

pupil
a vector of class numeric; the status/validity of the pupil
samplerate
the samplerate of the eyetracker
pad
time in ms to pad each side of each no pupil segment

Value

a vector of class logical; indicates samples that belong to a blink

Description

Detects blinks based on pupil status/valicity

Examples

data(lc120)
d.b <- with(lc120, detect_blinks.SW(status, 120))
str(d.b)
