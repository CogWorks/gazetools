Get Fixations

Usage

getFixations(class, dpva, drop = T)

Arguments

class
an object of class classify
dpva
an object of class pva
drop
a boolean, if T (default) drop fixations at (0,0)

Get Fixations

Description

Extracts the coordinates of fixations and their durations from classify and pva objects

Examples

data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl,
                       500, 1680, 1050, 473.76, 296.1,
                       smi_ezl, smi_exl, smi_eyl))
d.c <- classify.V(d.pva@v)
d.f <- getFixations(d.c, d.pva)
d.f
