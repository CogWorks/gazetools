Gaze Data Classification (velocity-acceleration)

Usage

classify.VA(v, a, vt = 30, at = 8000)

Arguments

v
a vector of instantaneous velocity values
vt
a velocity threshold
a
a vector of instantaneous acceleration values
at
an acceleration threshold

Value

an object of class classify

Description

Classifies gaze data into saccades and fixations using both velocity and acceleration thresholds.
