Gaze Data Classification (velocity-acceleration)

Classifies gaze data into saccades and fixations using both velocity and acceleration thresholds.

Usage

classify.VA(v, a, vt = 30, at = 8000, blinks = NULL)

Arguments

v
a vector of instantaneous velocity values
vt
a velocity threshold
a
a vector of instantaneous acceleration values
at
an acceleration threshold
blinks
vector of class "logical", TRUE if samples belong to a blink

Value

an object of class classify

References

NystrÃ¶m, M., & Holmqvist, K. (2010). An adaptive algorithm for fixation, saccade, and glissade detection in eyetracking data. Behavior Research Methods, 42(1), 188-204.

Examples

# Classification ignorning blinks
data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl,
                       500, 1680, 1050, 473.76, 296.1,
                       smi_ezl, smi_exl, smi_eyl))
d.c <- classify.VA(d.pva@v, d.pva@a)
str(d.c)
