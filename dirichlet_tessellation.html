Dirichlet Tessellation

Usage

dirichlet_tessellation(x, y, xrange, yrange)

Arguments

x
Vector of x coordinates of data points
y
Vector of y coordinates of data points
xrange,yrange
specify a window of observation with these dimensions

Dirichlet Tessellation

Description

Calculates the dirichlet tessellation of a set of points

Examples

data(smi)
d.pva <- with(smi, pva(smi_sxl, smi_syl,
                       500, 1680, 1050, 473.76, 296.1,
                       smi_ezl, smi_exl, smi_eyl))
d.c <- classify.V(d.pva@v)
d.f <- getFixations(d.c, d.pva)
d.tess <- dirichlet_tessellation(d.f$fixation.x, d.f$fixation.y, c(315,1365), c(0,1050))
str(d.tess)
