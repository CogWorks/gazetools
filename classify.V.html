Gaze Data Classification (velocity)

Usage

classify.V(v, vt = 75)

Arguments

v
a vector of instantaneous velocity values
vt
a velocity threshold

Value

an object of class classify

Description

Classifies gaze data into saccades and fixations using a velocity threshold.
